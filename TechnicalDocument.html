    Python while Loops and user inputs.

    user inputs

    how the user input works

    The input() function pauses your program and waits for the user to enter some text. Once Python receives the user’s input, it stores it in a variable
    to make it convenient for you to work with. For example, the following program asks the user to enter some text, then displays that message back to the user: message = input("enter your name: ") print(message) The input() function takes one argument: the prompt, or instructions, that we want to display to the user so they know what to do. In this example, when Python runs the
    first line, the user sees the prompt Tell me something, and I will repeat it back to you: . The program waits while the user enters their response and continues after the user presses enter.
    The response is stored in the variable message, then print(message) displays the input back to the user: enter your name: Nakayiza Shamim Nakayiza Shamim

    while Loops

    You can use a while loop to count up through a series of numbers.
    For example, the following while loop counts from 1 to 5: current_number = 1 while current_number <= 5: print(current_number) current_number += 1 In the first line, we start counting from 1 by setting the value of current_number to 1.
    The while loop is then set to keep running as long as the value of current_number is less than or equal to 5.
    The code inside the loop prints the value of current_number and then adds 1 to that value with current_number += 1.
    (The += operator is shorthand for current_number = current_number + 1.) outPut of the code above: 1 2 3 4 5
